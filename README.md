# Shoreline Design 🌊

> *Forever furniture, rooted in Nicaragua. Now available in Canada.*

Shoreline Design is a Canadian import and retail brand bringing [MasayaCo](https://www.masayaco.com)'s handcrafted, FSC-certified teak furniture to Canada. Every piece is made by skilled Nicaraguan artisans using responsibly harvested teak wood — designed to be heirloom quality, repairable, and built to last a lifetime.

This repository contains the brand launch assets, marketing materials, and trade program documents for Shoreline Design's Canadian market entry.

## 🪑 The Product

MasayaCo has been crafting furniture in Masaya, Nicaragua since 2014. Their pieces are:

- **FSC-certified** — wood grown in their own reforestation projects (1M+ trees planted)
- **Handwoven** — every pattern is named, hand-dyed, and woven by skilled artisans
- **Heirloom quality** — designed to be repaired, not replaced
- **Award-winning** — recognised globally for sustainable design

Shoreline Design is the exclusive Canadian importer, offering white-glove delivery across Canada.

## 🚀 Getting Started (Developers)

To run locally, simply open any `.html` file in your browser. No build tools or dependencies required — all files are self-contained HTML/CSS.

To deploy to GitHub Pages:

1. Rename your main landing page to `index.html`
2. Ensure all image files (`img1.jpg` – `img10.jpg` etc.) are in the same directory
3. Go to **Settings → Pages → Source → Main branch**
4. Your site will be live at `https://yourusername.github.io/shoreline-design`

To connect a custom domain (e.g. `shorelinedesign.ca`):
1. Purchase your domain from Namecheap or Google Domains
2. In GitHub Pages settings, add your custom domain
3. Update your domain's DNS to point to GitHub's servers
